# Habit Tracker (Spring Boot)
A local habit tracking application built with Spring Boot, Thymeleaf, and H2/SQLite.
This project helps track daily habits, completion streaks, and personal growth.

## Tech Stack
- Java 17
- Spring Boot 4
- Spring Data JPA
- H2 (Development)
- SQLite (Final)
- Thymeleaf

## Features
- Create, update, delete habits
- Mark daily completion
- Track streaks
- Local-only with web interface
- Future: reminders, reports, categories
